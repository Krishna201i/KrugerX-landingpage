Installation Guide

1

Download the Installer

Click the Download .exe (x64) button above. The file KrugerX-Setup-1.0.0-x64.exe (~128 MB) will begin downloading to your Downloads folder.

2

Run the Installer

Double-click the downloaded file. Windows UAC will ask "Do you want to allow this app to make changes?" — click Yes to continue.

If Windows Defender SmartScreen shows a blue "Windows protected your PC" dialog, click More info → Run anyway. KrugerX is safe — this appears because the app is new and not yet in Microsoft's catalog.
3

Choose Install Location

The installer defaults to C:\Program Files\KrugerX\. We recommend keeping the default. Click Install to proceed. Installation takes approximately 30 seconds.

4

Launch KrugerX

After installation completes, KrugerX opens automatically. You'll find it in your Start Menu and a shortcut on your Desktop. The app also appears in your system tray when minimized.

5

Complete Onboarding

The first-launch wizard lets you: import bookmarks from Chrome/Edge/Firefox, set your privacy level (Standard, Strict, or Maximum), choose your default AI model, and optionally create a KrugerX account for cross-device sync.

You can skip account creation and continue as a guest. Sync features will be unavailable, but all local browsing features work fully offline.

Installation Guide

1

Download the .dmg

Select the correct build: Apple Silicon for M1/M2/M3 Macs (recommended for 2020+ Macs), or Intel for older Intel Macs. The file is approximately 145 MB.

2

Open the Disk Image

Double-click the downloaded .dmg file. A window appears showing the KrugerX icon and your Applications folder. Drag KrugerX into the Applications folder to install it.

3

Handle macOS Gatekeeper

The first time you open KrugerX, macOS will block it with a message about an unidentified developer. To bypass this:

  1. Go to System Settings → Privacy & Security
  2. Scroll down to find the KrugerX entry
  3. Click "Open Anyway"
Alternatively, right-click (or Control-click) the KrugerX app in Finder → select Open → click Open in the dialog.
4

Grant Permissions

KrugerX will request: Notifications (for sync alerts and AI responses), Accessibility (for keyboard shortcuts), and optionally Full Disk Access for bookmark import. Each permission prompt can be accepted or skipped.

5

Complete Onboarding & Eject the .dmg

Complete the first-launch setup wizard. Once done, you can eject the disk image (drag it to Trash or right-click → Eject in Finder) and delete the original .dmg file from your Downloads folder.

Frequently Asked Questions

Is KrugerX really free?

Yes. KrugerX is free forever with no hidden fees. AI model usage may require API keys from the respective providers (OpenAI, Anthropic, etc.), but all core browser features are completely free.

Is KrugerX open source?

Yes. KrugerX is open source under the MIT License. You can view, fork, and contribute on our GitHub repository.

Does KrugerX collect my data?

No. KrugerX uses zero-knowledge encryption for all sync data. We cannot read your browsing history, bookmarks, or passwords. There is no telemetry by default.

Can I import from Chrome or Firefox?

Yes. The first-launch wizard lets you import bookmarks, history, saved passwords, and extensions from Chrome, Firefox, Edge, and Safari with one click.

Do I need an account to use KrugerX?

No. You can use KrugerX fully as a guest. An account is only required if you want to sync data across devices.

Which AI models are supported?

KrugerX integrates with OpenAI (GPT-4o), Google Gemini (2.5 Pro), Anthropic (Claude 4), Groq (Llama 3.1), and any local model via Ollama. You bring your own API keys.
